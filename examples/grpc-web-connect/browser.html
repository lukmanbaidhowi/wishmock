This example uses the gRPC-Web protocol to communicate with Wishmock via Connect RPC. Unlike the Connect protocol which uses JSON, gRPC-Web uses binary protobuf encoding with base64 for browser compatibility. Connect RPC natively supports gRPC-Web without requiring Envoy proxy!

Configuration

Note: Make sure Wishmock is running with Connect RPC enabled (CONNECT_ENABLED=true)

1. Unary RPC - SayHello (gRPC-Web)

Simple request-response call using gRPC-Web protocol to helloworld.Greeter/SayHello.

2. Server Streaming RPC - GetMessages (gRPC-Web)

Server streaming call using gRPC-Web protocol to streaming.StreamService/GetMessages.

3. Protocol Comparison

Compare the same request using both Connect (JSON) and gRPC-Web (binary) protocols.
